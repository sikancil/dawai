# API Reference: `@arifwidianto/dawai-common`

This section provides a detailed API reference for the `@arifwidianto/dawai-common` package. This package contains core utilities, decorators, interfaces, and type definitions used across the Dawai framework.

## Overview

`@arifwidianto/dawai-common` includes:
*   **Decorators**:
    *   Class Decorators (e.g., `@webservice()`, `@stdio()`, `@mcpServer()`, `@a2aAgent()`)
    *   Method Decorators (e.g., `@Crud()`, `@Cli()`, `@Ws()`, `@Sse()`, `@Rpc()`, `@Llm()`, `@Mcp()`, `@A2a()`)
    *   Parameter Decorators (e.g., `@Body()`, `@Query()`, `@Params()`, `@Headers()`, `@Cookies()`, `@Session()`, `@Files()`, `@Ctx()`, `@Req()`, `@Res()`)
*   **Interfaces**:
    *   `MicroserviceOptions` and its sub-options (e.g., `WebserviceOptions`, `StdioOptions`).
    *   Transport-specific context interfaces.
    *   Metadata interfaces (e.g., `HandlerMetadata`, `ParameterMetadata`).
*   **Constants**: Metadata keys and other shared constants.
*   **Utilities**: Helper functions used by the framework.
